#hello-go

##For Ubuntu16.04 server

1.install by apt-get
```
sudo apt-get install goloang-go
```
2.config go env
```
sudo vim vim /etc/profile
export GOROOT=/usr/lib/go-1.6
export GOPATH=$HOME/go-path
```
3.go cmd
```
$ go run xxx.go
$ go test
$ go test -v
$ go test -v -test.run 

$ sudo apt-get install golang-golang-x-tools
$ godoc -http=:8083
```

$ go version
